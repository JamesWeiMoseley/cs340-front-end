Orders Table
Description

A composition entity that records the details for each and every order such as who handled the order, who ordered it, where/when ordered, and the total price.

Items Table
Description

An entity that has all the possible menu items stored such as different sides, drinks, and entrees.

Customers Table
Description

An entity that keeps track of every customer who makes an order at Burger Basket and keeps some of their personal information such as email.

Employees Table
Description

An entity that has all the personal information cashiers of the restaurant such as name, email, and username.

Cooks Table
Description

An entity that has all the personal information cooks of the restaurant such as name, email, and username.
